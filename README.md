# Start React.js

- Scoop 설치
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -scope CurrentUser
```
```powershell
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
```

- git, aria2 설치
```powershell
scoop install git aria2
```

- touch, chrome 설치
```powershell
scoop install touch
```
